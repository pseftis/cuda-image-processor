Performance Analysis
====================
- The CUDA implementation shows consistent high throughput across different image sizes
- Larger images demonstrate better GPU utilization (higher throughput)
- Processing time scales approximately linearly with image size
- Larger kernel sizes increase computation per pixel, slightly reducing throughput
- The implementation successfully handles images up to 4096x4096 (16.7 megapixels)
